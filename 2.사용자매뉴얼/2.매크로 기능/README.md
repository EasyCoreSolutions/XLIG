
# 매크로 기능

XLIG는 SQL이 제공하는 기능과 엑셀 기능을 한번에 사용해 데이터를 핸들링하는 환경을 제공하며, 엑셀 매크로 기능을 사용해 SQL을 동적으로 활용할 수 있습니다. 사용자는 VBA를 활용하지 않고도 XLIG가 제공하는 매크로를 사용해 쉽게 동적인 대시보드를 구성할 수 있습니다.
